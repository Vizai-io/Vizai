# Customer Profile Template

Use this folder as a starter package for a new customer under `businesses/{customer-slug}/`.

Recommended files:
- `profile.json`
- `profile.jsonld`
- `company.md`
- `services.md`
- `products.md`
- `locations.md`
- `industries.md`
- `faqs.md`
- `proof-points.md`
- `sources.json`
- `approval.json`
- `changelog.md`

Replace all placeholder values and remove sample-only language before production use.
