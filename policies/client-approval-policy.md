# Client Approval Policy

All publishable customer profile data should be approved before external use.

## Approval Steps
1. Prepare customer profile package.
2. Attach source evidence for key claims.
3. Submit approval packet.
4. Record decision in `approval.json`.
5. Publish only approved scope.

## Decision States
- `pending`
- `approved`
- `rejected`

## Recordkeeping
Approval metadata should include approver identity, date, and approved scope.
