# Repo Architecture

This repository is the rich customer data hub for VizAI.

## Core Areas
- `businesses/`: per-customer rich profile packages
- `schema/`: JSON Schema contracts for profile artifacts
- `templates/`: starter templates for onboarding and approvals
- `scripts/`: validation and build automation
- `policies/`: governance and publication standards
- `docs/`: operational guides and architectural notes

## Design Intent
Keep rich customer truth in this repo, while supporting export to a separate lightweight business registry index.
