# Onboarding Workflow

## Goal
Create a rich, validated, AI-readable customer profile package.

## Steps
1. Copy template assets for a new `businesses/{customer-slug}/` folder.
2. Collect customer facts, offerings, locations, and approved proof points.
3. Add source records to `sources.json`.
4. Build `profile.json` and `profile.jsonld`.
5. Run validation scripts.
6. Submit approval packet and record `approval.json`.
7. Publish after approval.
